## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript + MongoDB starter repository.
Implemented the base controller and service to avoid the CRUD related duplication code and made generic controller and service implementaitons

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
